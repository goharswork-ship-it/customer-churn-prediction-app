# Customer Churn Prediction App

A full-stack machine learning application that predicts whether a telecom customer is likely to churn.

## Features
- Customer churn prediction
- Random Forest classification
- Churn probability score
- Risk level and retention guidance
- FastAPI backend
- React dashboard
- Responsive UI

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- FastAPI
- React
- Vite
- HTML
- CSS
- JavaScript
