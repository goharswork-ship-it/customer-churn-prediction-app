\# Customer Churn Prediction App



A full-stack Machine Learning application that predicts whether a telecom customer is likely to churn.



\## Tech Stack



\- Python

\- Scikit-learn

\- Pandas

\- NumPy

\- FastAPI

\- React

\- Vite

\- HTML

\- CSS

\- JavaScript



\## Features



\- Customer Churn Prediction

\- Random Forest Classifier

\- Churn Probability

\- Risk Level

\- Responsive Dashboard

\- FastAPI REST API
